\# Software Quality Evaluation System



\## Description

This project evaluates the quality of the ShopSphere software system using selected ISO/IEC 25010 quality characteristics and ISO/IEC 25023 metrics.



The user enters measured metric values, and the program calculates:

\- criterion scores

\- dimension scores

\- overall quality score

\- weakest quality dimension



\## System Evaluated

ShopSphere (Web System)

Version: 3.2.1



\## Quality Dimensions

\- Functional Suitability

\- Reliability

\- Performance Efficiency

\- Maintainability



\## Technologies Used

\- Java

\- OOP

\- ArrayList
